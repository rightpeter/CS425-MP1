# CS425-MP1

## How to run this project

In /CS425-MP1 folder:

## Build

Use

	go build -o server.bin ./server && go build -o client.bin ./client

to build excecutable files.

## Run

Server:

	./server.bin -p <PORT_NUMBER> -IP <IP_ADDRESS>

	// IP flag is currently optional

Client:

	./client.bin
